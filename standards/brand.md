# Gaudi AI Website Instructions

This project is the marketing website for Gaudi AI.

Gaudi AI is the AI operating layer for modern builders. It helps construction teams automate back-office workflows including estimating, bid leveling, proposal generation, document review, project updates, and operational admin.

## Audience

- General contractors
- Estimators
- Project managers
- Construction executives
- Developers and owners
- Trade contractors

## Brand feel

The website should feel:

- Premium
- Modern
- Construction-native
- Operational
- Trustworthy
- Minimal but not empty
- AI-enabled but not generic AI hype

## Design direction

Use:

- HEX: #c35a25 for the logo and highlights
- Thin borders
- Wherever applicable, document cards
- Wherever applicable, workflow diagrams
- Wherever applicable, product UI mockups

Avoid:

- Generic AI gradients
- Stock construction photos as the main hero visual
- Cartoon construction workers
- Chatbot-only visuals
- Sci-fi styling
- Fake customer logos
- Made-up metrics unless clearly labeled as estimates

## Build rules

- Organize the website into separate, easy-to-edit components for each section: Navbar, Hero, Mission, HowItWorks, Benefits, ROICalculator, Testimonials, BackedBy, About, Events, FinalCTA, and Footer.
- Use Tailwind CSS to style the website, including colors, spacing, typography, buttons, cards, and responsive layouts.
- Make the site responsive.
- Keep copy specific to construction workflows.
- Do not overdo animations.
- Do not make the product look like a generic chatbot.
